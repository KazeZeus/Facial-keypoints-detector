Facial Keypoint detction

This project is built on the principal that our face has many keypoints that differentiates it from rest of the image.

Thus this model detects those keypoints from the images and is able to detect faces in other images.

architure is of a CNN with 5 Convulational layers implemented.
Amazon aws data is used in buiolding the data base.

KEY TECHNOLIGIES 
 pytorch 
and opencv
